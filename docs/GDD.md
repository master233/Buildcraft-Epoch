# BuildcraftEpoch — 游戏设计文档 (GDD)

## 游戏类型
2D 模拟经营 + 肉鸽养成

## 目标平台
**HTML5** — 浏览器直接运行，无需安装

---

## 建筑系统

### 通用规则
- 共 6 栋建筑，每栋分为 **3 个等级**
- 升级消耗：**木材 + 矿石**
- **主基地等级**限制其他所有建筑的最高等级

### 建筑列表

| 建筑 | 作用 |
|------|------|
| 主基地 | 核心建筑，等级限制其他建筑上限，解锁新功能 |
| 伐木场 | 定时产出木材，等级越高产量越多 |
| 矿石场 | 定时产出矿石，机制与伐木场相同 |
| 酒馆 | 刷新并购买角色，等级越高刷出高品质角色概率越高 |
| 研究院 | 研究流派或装备的养成路线 |
| 远征塔 | 塔层结构，每层一个 Boss 关卡，通关可获得装备等奖励 |

## 角色系统

### 升星机制
- 3 个相同角色合并 → 升 1 星
- 每次升星可强化角色的属性和技能

### 技能
- **主动技能**：魔法值充满后战斗中自动释放
- **被动技能**：持续生效

### 属性
| 属性 | 说明 |
|------|------|
| 攻击力 | 造成伤害 |
| 护甲 | 减少受到的伤害 |
| 闪避 | 规避攻击的概率 |
| 暴击 | 触发暴击伤害的概率 |
| 吸血 | 攻击时回复生命 |
| 回蓝速度 | 魔法值恢复速度 |
| 攻击速度 | 攻击频率 |
